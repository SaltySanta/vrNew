Detailed Description

Overloaded control class for the VR around a memory game.

  • Derived from BehaviourScript
  • Spawns pairs of memory-objects covered with boxes in a table
  • Processes the game logic of the memory game
  • Implements a simple AI which can be played against
  • Intended to be used with the "Memory" scene
  • Created on: 2015
  • Author: Sascha Jüngel
