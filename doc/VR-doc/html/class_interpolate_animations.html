Detailed Description

Class for interpolating/blending 3 grasp animations to grasp an intermediate position. This class is finding the smallest triangle to blend animation.

Author
Onur Karaduman (onrkr.nosp@m.dmn@.nosp@m.gmail.nosp@m..com), Frederik beuth(beuth.nosp@m.@hrz.nosp@m..tu-c.nosp@m.hemn.nosp@m.itz.d.nosp@m.e)
  • Created on: August 2013
  • Idea:
    • Class stores a list of existing animations and their final coordinates.
    • It constructs from this point list a list of triangles.
    • It then receives a target point to grasp and find the smallest triangle containing this point (see animate(...) ).
    • Then, it calculates euklidean distance from the target to the trianlge vertex (corner points).
    • These values are used as factors to blend 3 animations together.
