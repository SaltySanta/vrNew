Detailed Description

This class controls the whole VR.

It assigns network interfaces to the agents, and has an own network interface to receive commands that concern the running trail. It is not intended that this class is directly attached to a 3D Object in Unity. Therefore the user must create a derived class for his scenario, where the behavior to protobuf messages can be overridden.

  • Created on: November 2011
  • Author: Marcel Richter
  • Modified: September 2012
  • Author: Michael Schreier

Responsibility of this class

  • Network object creation
  • Spans agents
  • Create and delete barriers
  • GUI/Button handling (BehaviourScript::OnGUI)
  • Actions
  • Save camera images to disk for Debug purposes (BehaviourScript::SaveScreenPNG)
