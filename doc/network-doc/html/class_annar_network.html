Detailed Description

Together with AnnarLayer, AnnarNeuron, AnnarWeightList and AnnarWeight, AnnarNetwork, this is the data class for a neuronal network.

The neuronal network is organized into layers, neurons and weights. AnnarNetwork holds the list of neuronal layers (AnnerLayer) and provides access to these. Each layers holds then their neurons (AnnerNeuron) which in turn stores the weights (AnnarWeightList and AnnerWeight). In summary, the Annar* - casses provide a nested data structure to store a neuronal network.
